# Редактор деревьев

Веб-приложение для редактирования иерархических структур данных (деревьев) с двухпанельным режимом.

## Возможности

- Двухпанельный режим для сравнения деревьев
- Добавление/удаление узлов
- Drag & drop между деревьями
- Вырезать/копировать/вставить
- Импорт/экспорт в XLSX
- Поиск с подсветкой результатов
- Настройка динамических атрибутов
- Виртуальный скроллинг для больших деревьев
- Inline-редактирование полей

## Структура файлов

```
index.html    - HTML разметка
styles.css    - CSS стили
app.js        - JavaScript логика
```

## Запуск

Откройте `index.html` в браузере.
